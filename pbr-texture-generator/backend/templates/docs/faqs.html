FAQs & Common Issues

Find answers to common questions and solutions to potential issues while using MapMorph.

Frequently Asked Questions

1. What is MapMorph?

MapMorph is a web-based tool that converts real-world images into PBR textures for use in 3D applications, game engines, and rendering software.

2. Do I need to install anything?

No, MapMorph runs entirely in your web browser—no installation required!

3. What image formats are supported?

Currently, MapMorph supports JPG, PNG, and TIFF image formats.

4. Can I use the generated textures in commercial projects?

Yes! You are free to use MapMorph-generated textures in both personal and commercial projects.

5. How do I export my textures?

After processing, you can download textures in PNG format. Additional formats may be added in future updates.

Common Issues & Solutions

1. My textures are generating slowly.

Processing times depend on the image size and complexity. Try using a lower resolution image for faster results.

2. The generated normal map looks incorrect.

Ensure the image has clear surface details and good lighting. You can also tweak the normal intensity settings.

3. The website isn’t loading properly.

Try clearing your browser cache or using a different browser. MapMorph works best on Chrome, Firefox, and Edge.

4. My downloaded textures are missing.

Check your browser’s download folder and ensure pop-ups are enabled for MapMorph.

5. Where can I report a bug?

If you encounter an issue, please report it on our GitHub Issues page or contact support.
