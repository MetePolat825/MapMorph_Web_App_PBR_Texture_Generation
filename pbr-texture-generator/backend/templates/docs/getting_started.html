Getting Started with MapMorph

Welcome to MapMorph! Follow these steps to start generating high-quality PBR textures from real-world images.

1. Accessing MapMorph

MapMorph is a web-based tool, so there is no need for installation. Simply open your preferred browser and navigate to: https://mapmorph.com

2. Uploading an Image

Click on the "Upload Image" button and select a high-resolution image of a surface you want to convert into PBR textures.

3. Choosing a Workflow

MapMorph supports multiple PBR workflows:

  • Roughness/Metallic - Used in engines like Unreal Engine.
  • Specular/Glossiness - Common for Unity and older engines.
  • Bump Mapping - For simpler shading models.

4. Adjusting Settings

Once the image is uploaded, tweak parameters like:

  • Intensity of Normal Maps
  • Roughness Strength
  • Metallic Reflection

5. Generating & Downloading Textures

Click "Generate Textures". Once processed, you can preview the results and download maps in PNG or JPG format.

6. Using Your Textures

Import your generated textures into game engines or 3D software like Blender, Unity, or Unreal Engine.
