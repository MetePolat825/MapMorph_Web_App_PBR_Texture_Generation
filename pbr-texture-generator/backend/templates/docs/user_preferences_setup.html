User Preferences

MapMorph provides a range of customization options to help users tailor their texture generation workflow. This page covers all available preferences, including export configurations, AI-assisted tools, and processing settings.

Export Configuration

Customize how your textures are saved and structured upon export.

  • File Format: Choose between PNG, TGA, BMP, or JPG.
  • Resolution: Options range from 512x512 up to 4096x4096 (4K).
  • Bit Depth: 8-bit or 16-bit for higher precision.
  • Export Naming Scheme: Customize file names for easier organization.
  • Batch Export: Enable/disable exporting multiple textures at once.

PBR Presets

MapMorph allows you to configure PBR values to match different rendering engines.

  • Standard Presets: Choose from default settings optimized for Unity, Unreal Engine, or Blender.
  • Custom Presets: Upload or define your own PBR values for a more customized workflow.
  • Preset Switching: Easily toggle between different workflows (Roughness/Metallic vs. Specular/Glossiness).

AI-Assisted Tools

MapMorph integrates AI-powered enhancements for texture generation.

  • Smart Material Segmentation: AI detects and categorizes different surface materials within an image.
  • Auto Normal Map Refinement: Enhances normal map details for improved depth and accuracy.
  • Seamless Texture Tiling: AI-assisted tiling ensures smooth texture repetition.
  • Color Consistency Correction: Balances lighting and color to ensure uniform texture appearance.

Processing Preferences

Customize how textures are processed before final export.

  • Detail Intensity: Adjust how much fine detail is preserved in the output.
  • Noise Reduction: Enable AI-powered noise filtering to remove unwanted grain.
  • Texture Scaling: Upscale or downscale textures dynamically.
  • Sharpening & Contrast: Fine-tune texture sharpness and contrast levels.

UI & General Preferences

Configure the interface and workflow for a better user experience.

  • Dark Mode: Toggle between light and dark themes.
  • Auto-Save: Automatically save preference configurations.
  • Keyboard Shortcuts: Enable quick-access hotkeys for faster workflow.
  • Language Selection: Choose from multiple UI language options.

Saving & Loading User Preferences

MapMorph allows users to save and reload their preferences for a seamless workflow.

  • Save Preferences: Store your custom settings for future sessions.
  • Load Preferences: Reload saved configurations at any time.
  • Reset to Default: Revert settings to the original factory defaults.
