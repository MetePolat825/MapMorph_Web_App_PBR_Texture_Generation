Contributing to MapMorph

We welcome contributions from the community! Whether it's bug reports, feature requests, or direct code contributions, your help is invaluable in improving MapMorph.

Ways to Contribute

1. Reporting Issues

Found a bug? Have a suggestion? Please open an issue on our GitHub repository.

Report an Issue on GitHub

2. Requesting Features

Have an idea for an improvement? We encourage feature requests! Please describe the enhancement you'd like to see.

3. Submitting Code Contributions

MapMorph is an open-source project. If you'd like to contribute code, follow these steps:

  1. Fork the repository.
  2. Create a new branch for your changes.
  3. Make your changes and test them.
  4. Submit a pull request.

See our contribution guidelines for more details.

4. Translating Documentation

Help make MapMorph accessible to a wider audience by translating our documentation into different languages.

5. Spreading the Word

Share MapMorph with fellow developers, 3D artists, and game designers. The more people who use it, the better it gets!

Community and Support

Join the MapMorph community for discussions and support:

  • Join our Discord
  • Follow us on Twitter
  • Check out our GitHub

Thank you for contributing to MapMorph!
