Uploading Textures & Setup

Before generating PBR textures in MapMorph, you need to upload an image. This page covers the supported formats, file size limitations, and common issues you might encounter.

Supported File Formats

Currently, MapMorph supports the following image formats:

  • TGA (.tga)
  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • BMP (.bmp)

Support for additional formats (e.g., TIFF, EXR) is planned for future updates.

File Size & Resolution Limits

To ensure smooth processing, your uploaded texture must meet these requirements:

  • Minimum resolution: 256 x 256 pixels
  • Maximum resolution: 4096 x 4096 pixels (4K)
  • Maximum file size: 10MB

Files that are too small, too large, or exceed the size limit will not be processed.

Uploading Process

Follow these steps to upload your texture:

Step 1: Select an Image

  • Click the Upload button.
  • Select a valid texture file from your device.
  • Ensure the file meets the supported format and size requirements.

Step 2: Verify Your Upload

  • After uploading, a preview of your texture will appear.
  • If the texture is incorrect or needs adjustments, you can click Reupload to select a different file.

Step 3: Proceed to Processing

  • Once satisfied with the uploaded image, click Process Texture.
  • MapMorph will begin analyzing the image and generating PBR texture maps.

Common Upload Issues & Solutions

❌ File Format Not Supported

Issue: "Unsupported file format" error appears.

Solution: Convert the image to a supported format (TGA, JPG, PNG, BMP) before uploading.

❌ Image Too Large or Too Small

Issue: Image resolution is below 256x256 or exceeds 4K.

Solution: Resize the image within the allowed range using an image editor.

❌ File Too Large

Issue: "File exceeds 10MB limit" error.

Solution: Compress the image or reduce resolution before uploading.

❌ Upload Stuck or Not Responding

Issue: Upload progress bar gets stuck.

Solution: Refresh the page, ensure your internet connection is stable, and try again.
