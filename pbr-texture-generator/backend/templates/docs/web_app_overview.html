Web App Overview

MapMorph is a powerful web-based tool designed to simplify the process of generating PBR textures from real-world images. This page provides a high-level overview of the application's core features and functionality.

Key Features

  • AI-Powered Texture Generation: Convert real-world images into PBR textures with a single click.
  • PBR Preset System: Ensure texture consistency across different rendering engines with predefined and custom PBR presets.
  • Preference Management: Save and load user settings for a streamlined workflow.
  • Cross-Platform Compatibility: Works in the browser with no installation required.
  • Real-Time Previews: Visualize texture maps before exporting.
  • Support for Multiple Workflows: Roughness/Metallic and Specular/Glossiness workflows are available.

Logging In & User Accounts

MapMorph supports both guest usage and user accounts. Logging in provides additional features such as:

  • Saving and loading texture preferences
  • Accessing cloud storage for previously generated textures
  • Syncing settings across devices

To create an account, simply click on the Sign Up button on the homepage.

Saving & Loading Preferences

Users can save custom settings, including PBR presets, texture resolution, and output format. These settings can be loaded in future sessions for efficiency.

How to Save Preferences:

  1. Navigate to the Settings menu.
  2. Adjust the parameters as needed.
  3. Click the Save Preset button and assign a name.

Loading a Saved Preset:

  1. Open the Settings menu.
  2. Select a saved preset from the dropdown list.
  3. Click Load to apply the settings.

Understanding PBR Presets

PBR presets help ensure that generated textures adhere to industry standards. MapMorph includes built-in presets for:

  • Unity
  • Unreal Engine
  • Custom user-defined presets

Users can also upload their own PBR configuration files for maximum flexibility.

Using MapMorph: Basic Workflow

To generate textures using MapMorph, follow these simple steps:

Step 1: Upload an Image

  • Click the Upload button and select a JPG, PNG, or TIFF file.

Step 2: Adjust Processing Settings

  • Choose the PBR workflow (Roughness/Metallic or Specular/Glossiness).
  • Enable optional AI-based material segmentation.

Step 3: Generate & Preview Textures

  • Click Generate to process the image.
  • Use the built-in viewer to inspect normal, roughness, and albedo maps.

Step 4: Export Textures

  • Download textures in PNG format.
  • Save the generated maps for future use.
